<h3 id="project1">Project 1: AI Chest X-ray Disease Detection System</h3>

<p>

The AI Chest X-ray Disease Detection System is a deep learning based medical imaging project designed to assist healthcare professionals in identifying thoracic diseases from chest X-ray images. The system utilizes a YOLO based object detection model to locate abnormalities within radiographic images and highlight affected regions using bounding boxes.

The system includes an interactive Streamlit web interface that allows users to upload chest X-ray images, adjust visualization parameters, and view the model’s predictions in real time. Detected findings are displayed visually along with confidence scores, enabling quick interpretation.

To enhance clinical usability, the system integrates a locally hosted language model which generates structured medical style reports based on the detected findings. This provides a simulated radiology-style interpretation of the scan results.

The project demonstrates how artificial intelligence can assist in medical image analysis by combining computer vision models, web application frameworks, and automated reporting systems into a single workflow.

</p>
