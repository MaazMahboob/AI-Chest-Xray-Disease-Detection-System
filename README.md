<h3 id="project1">Project 1: AI Chest X-ray Disease Detection System</h3>

<p>

The AI Chest X-ray Disease Detection System is a deep learning based medical imaging project developed to assist in identifying thoracic diseases from chest X-ray images. The system uses a YOLO based object detection model to localize abnormalities and highlight affected regions in radiographic images.

An interactive Streamlit application was developed to allow users to upload X-ray images, visualize detection results with bounding boxes, and analyze model predictions in real time. The system also integrates a locally running language model to generate structured clinical style reports from the detected findings.

This project demonstrates the integration of computer vision, deep learning, and clinical reporting tools to build an end-to-end AI assisted diagnostic workflow for medical imaging analysis.
